Matlab code of paper "ADAPTIVE DATA-DRIVEN PROBABILISTIC REDUCED-ORDER1
MODELS FOR PARAMETERIZED DYNAMICAL SYSTEMS", submitted to SIAM journal on Scientific Computing.



The following function is used to Construct reduced order model (ROM):

Burger_ROM: Burger equation 

Redi_ROM:  reaction-diffusion equation

Redi_10d_ROM: 10-d reaction-diffusion equation

NS_ROM:  N-S equation




The following function is used to develop adaptive sampling for constructing parametric reduced order model (pROM) for:

Burger_Adaptive_ROM: Burger equation

Redi_Adaptive_ROM:  2-d reaction-diffusion equation

Redi_10d_Adaptive_ROM: 10-d reaction-diffusion equation

NS_Adaptive_ROM_GPR: N-S equation
